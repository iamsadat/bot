# jargon/ — parked for disposal

This folder holds the **TradeBot** project (an unrelated algorithmic-trading
demo) that previously shared this repository with JobHunt. It has been moved
here, intact and with git history preserved (`git mv tradebot jargon`), so it
can be deleted at your convenience without touching JobHunt.

Nothing in JobHunt imports from this folder — it is fully self-contained and
safe to remove with `git rm -r jargon`.

Original entry point: `python -m jargon` (see `README.md` in this folder).
